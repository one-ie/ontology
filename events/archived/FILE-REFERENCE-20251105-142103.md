## Important Note

All file paths in this document are **absolute paths** from the system root (`/Users/toc/Server/ONE/`).

When working in development:
- Use relative imports: `import { DocList } from '@/components/docs/DocList'`
- The `@` alias resolves to `/web/src/`
- Build artifacts go to `/web/dist/`
- Content lives in `/web/src/content/`
